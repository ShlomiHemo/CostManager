**Full-Stack Expense Management Application**  
A personal finance tool that helps users track, manage, and analyze their expenses efficiently. Built with a focus on responsiveness, offline functionality, and data visualization.

---

## Features

- **Add, Edit & Delete Expenses:** Users can create new expense entries with category, description, price, and date.
- **Offline Storage:** Uses IndexedDB for persistent offline storage of all expense data.
- **Dynamic Table View:** Real-time updates of expense entries in a sortable and paginated table.
- **Reporting & Analytics:** Generate annual or monthly reports, visualized with interactive pie charts.
- **Collapsible Details:** Drill down into individual transactions within each expense category.
- **Form Validation:** Ensures all required fields are correctly filled before submission.

---

## Tech Stack

- **Frontend:** React, Material-UI, CSS Grid & Flexbox  
- **Backend / Storage:** IndexedDB (Offline Storage)  
- **Visualization:** React Google Charts, Collapsible Tables  
- **State Management:** React `useState` & `useEffect` hooks  
- **Development Tools:** Webstorm, Git

---

## Key Learnings & Skills

- Component-based architecture and state management in React.
- Integration of frontend with browser-based storage (IndexedDB) for offline functionality.
- Building dynamic, interactive data tables and charts for analytics.
- Applying UX principles to form validation and real-time user feedback.
- Handling CRUD operations efficiently and ensuring data persistence.
